# 🩺 MediLens AI

An AI-powered healthcare assistant that combines Machine Learning, Artificial Intelligence, and Retrieval-Augmented Generation (RAG).

## Features

- Disease Risk Prediction
- Medical Report Analysis
- Medical Document Chatbot
- AI Health Insights

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- PyMuPDF
